# rm
```
$ rm waterboy.txt
```
用 ``rm`` 命令移除文档和目录。这里我们从系统文件移除 **waterboy.txt** 。

```
$rm -r comedy
```
``-r`` 选项更改了 ``rm``的默认操作。 ``-r`` 的意思是 “recursive”,  用来移除一个目录和它的所有子集。

要小心使用 ``rm`` ! 它是永久性删除文档和目录的，没有恢复删除的命令，所以，一旦你用``rm`` 移除一个文档或者目录，就再也找不回来了。
