# **pwd** #

```
$ pwd

/**/**/blog
```
* 这里星号根据自己实际目录为准

``pwd`` 意思是 ‘print working directory(打印工作目录)’，会输出当前你所在的目录的名字，叫做工作目录。

当前的工作目录是 blog/。

和 ``ls`` 一起， ``pwd``命令通常是显示你在文件系统的具体位置。
