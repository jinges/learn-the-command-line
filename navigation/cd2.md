# **cd 2**

```
$ cd jan/memory
```
用 ``cd`` 带上一个目录的路径作为参数，可也指向到一个目录去。 ``cd jan/memory/`` 命令指向到了 jan/memory目录。

```
$ cd ..
```
用  ``cd ..`` 跳转到上级目录。 在这里  ``cd ..``  会从 jan/memory/指向到上级目录 jan/ 。 


用法
---

  1，切换目录到 2015/feb/目录

  ```
  $ cd ../feb
  ```

  列出当前工作目录里的所有的文件和目录。

  2，输入：
  ```
  $ mkdir media
  ```

  再一次列出当前工作目录的所有文件和目录。你会看到现在多了一个名叫 media/的目录。
