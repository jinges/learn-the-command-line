# **mkdir**

```
$ mkdir media
```

这里 ``mkdir``命令的意思是'新建目录'. 它会用一个目录的名字作为参数， 然后创建在当前工作目录里创建一个新的目录。

这里我们用  ``mkdir`` 在 feb/的目录里创建一个名叫 media/的目录。


用法
---

  1，指向到 2014/dec 目录
  ```
  $ cd ../../2014/dec
  ```
  列出当前工作目录里的所有文件和目录。

  2，接着输入：
  ```
  $ touch keyboard.txt
  ```

  再一次，列出当前目录里的所有文件和目录，你将会在哪里看到一个新的名叫 keyboard.txt的文件。
